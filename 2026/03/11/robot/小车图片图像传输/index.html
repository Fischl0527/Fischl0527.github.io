视频简单测试

同理，视频也是一样的。

windows

ubuntu

实际上，视频和图片的接收端可以完全一样，核心原因是传输协议完全一致：

  • 无论是图片还是视频，都使用相同的数据包结构：4字节长度头 + JPEG数据
  • 服务器只处理字节流，不区分内容是静态图片还是动态视频帧

关键优势：

  • 同一个 Windows 程序可接收图片、视频、摄像头流
  • 无需修改服务器代码，只需更换客户端
  • OpenCV 的 imdecode() 和 imshow() 对两者处理方式完全相同

本质：服务器只负责”接收数据→显示图像”，内容来源由客户端决定。
